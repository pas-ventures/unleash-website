The Concept

What is Paperclip?

If you've been working with Claude Code or OpenClaw, you've seen how fast things grow — and how messy they get. Paperclip wants to be the answer.

"If an AI agent is an employee,
Paperclip is the company."

The Idea

  • Open-source platform to run an agent-only company
  • Multiple AI agents, each with a defined role
  • Agents have jobs, budgets, accountability
  • Self-hosted. You own everything.

What You Can Do

  • Hire agents — CEO, Sales, Content, Community
  • Set budgets — monthly caps, auto-throttling
  • Track work — every task tied to goals
  • Stay in control — approve, pause, override
The Problem

Why Paperclip?

The tools are incredible — but they grow fast and get messy. Paperclip claims to have an answer.

🤖
Claude Code / Cursor

One agent at a time. 20 terminals. Context lost between sessions.

☁️
Cloud Agents

No organizational layer. No budgets, no accountability.

🔧
DIY Frameworks

Too technical. Most founders hit a wall at "now what?"

→

Paperclip Bridges the Gap

One dashboard. Multiple agents. Roles, budgets, goals. From playing with AI to agents running your business.
